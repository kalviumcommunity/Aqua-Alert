Function Data Flow Walkthrough

1. Defining Functions with Parameters

We start by defining a function that accepts input parameters (initial_level and final_level).

def calculate_water_level_change(initial_level, final_level):
    # 2. Using parameters inside the logic
    level_change = final_level - initial_level
    
    # 3. Returning a value using 'return'
    return round(level_change, 2)

2. Second Function for Reusing Data

This function takes the processed data to generate an output. Notice we use return to output the string.

def generate_alert_message(location, level_change):
    if level_change > 0:
        return f"ALERT: Water level at {location} has increased by {level_change} meters."
    else:
        return f"INFO: Water level at {location} remains stable."

3. Executing the Data Flow

We define some variables initially, and then call our functions: passing arguments, and storing the returned values in new variables.

# Defining variables
district_name = "Riverside"
morning_water_level = 15.5
evening_water_level = 16.8

# 4. Pass arguments and store the returned value
change_result = calculate_water_level_change(morning_water_level, evening_water_level)

# 5. Reuse the returned value meaningfully
alert_output = generate_alert_message(district_name, change_result)
